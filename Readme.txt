Название программы: AniTyan_Bot

Начало разработки: 15 ноября 2021(15.11.21)

Конец разработки: 20 декабря 2021(20.12.21)

Краткое описание программы: Это бот, который парсить расписание из excel файла

Язык и ПО: Python, SQL. SQLite, Sublime Text 3

Краткое описания всех скриптов:
   -(папка) .idea --> Автоматически созданная папка, языком Python
   -(папка) __pycache__ --> Автоматически созданная папка, языком Python
   -(папка) cisci --> Хранит в себе набор изображений с котиками
   -(скрипт) animania --> Прослушивает события в Telegram и отвечает в зависимости от сообщения
   -(скрипт) base_TG --> Взаимодействует с БД Telegram
   -(скрипт) base_VK --> Взаимодействует с БД VK
   -(скрипт) chat_VK --> Прослушивает события в VK и отвечает в зависимости от сообщения
   -(скрипт) config --> Хранит в себе токены и id бота
   -(скрипт) read_excel --> Парсит документ excel и формирует расписание
   -(БД) TG.db --> Хранит id чатов и имена преподавателей/учеников Telegram
   -(БД) vk_group.db --> Хранит id чатов и имена преподавателей/учеников VK
   -(Файл) Расписание.xlsx --> Расписание ВТК

Что было сделано: 
   - Бот может прослушивать сообщений и отвечать на них
   - Бот может читать excel документ и формировать расписание
   - Бот может использовать БД

Что не было добавлено:
   - Автоматическое скачивание расписания с сайта колледжа
   - Бот не отправляет мемы
   - Нет плююшек для преподаватля по практике Марии Денисовны
   - Нет ни одного секрета

Для чего разработано: Создание этой программы было практическим заданием в колледже
